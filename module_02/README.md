# GPS Lost Item Tracker

This project is a simple lost item tracker built with Python and SQLite. The purpose of the software is to help users keep track of important items they may lose, such as keys, a backpack, a wallet, or other personal belongings.

The program stores item information in a SQLite relational database. Users can add new lost items, view saved items, search for items, update an item's location or status, and delete items from the database. The project also includes a date range search feature so users can filter items based on when they were last seen.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Make sure Python is installed on your computer.
2. Download or clone this project from GitHub.
3. Open the project folder in VS Code or another code editor.
4. Run the program with this command:

```bash
python lost_item_tracker.py
